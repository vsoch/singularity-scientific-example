Container Analysis: HPC vs. Cloud

?

This project aimed to be a simple comparison of running a simple analysis pipeline on both cloud (AWS,GCE) and local compute (sherlock,scg4) resources. Each deployed the analysis by way of a Singularity container from Singularity Hub, and cloud resources that allow the user to be a superuser also deployed the equivalent analysis in Docker container. For more information about the analysis and runs, please see the code base on Github, and expect details and discussion in an upcoming manuscript.
